# Real_Estate_Agency_Data_Analysis
# Анализ данных агентства недвижимости

**Цель проекта**:  
Проанализировать объявления о продаже жилой недвижимости в Санкт-Петербурге и Ленинградской области, чтобы помочь агентству недвижимости из Петрозаводска разработать эффективную бизнес-стратегию при выходе на новый рынок.

## 🔧 Стек технологий
- PostgreSQL (DBeaver)
- DataLens 
 
## 📁 Данные  
Архивная информация сервиса «Яндекс Недвижимость» (схема real_estate) — объявления о продаже жилой недвижимости в Санкт-Петербурге и Ленинградской области за несколько лет.
